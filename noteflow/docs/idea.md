# NoteFlow

## ¿Qué problema resuelve?
El caos mental. Permite apuntar ideas y tareas rápido, sin menús ni configuraciones complejas que hagan perder el tiempo.

## Usuario y uso diario
*   **Quién:** Estudiantes y desarrolladores que necesitan un bloc de notas rápido.
*   **Uso:** Abrir, escribir la idea en tres segundos para que no se olvide, y cerrar.

## MVP (Versión 1)
*   Crear y borrar notas de texto rápido.
*   Marcar notas como pendientes o completadas.
*   Guardado automático en el móvil.
