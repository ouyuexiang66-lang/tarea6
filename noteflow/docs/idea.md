# NoteFlow - Definición de la Idea

## 1. Qué problema resuelve NoteFlow
En el día a día, las personas reciben un flujo constante de información dispersa: ideas fugaces, tareas pendientes, notas de reuniones o recordatorios visuales. Las herramientas actuales suelen ser o demasiado complejas (requiriendo mucha fricción para anotar algo rápido) o demasiado simples (dejando las notas enterradas en un historial sin organización).

**NoteFlow** resuelve la fragmentación y el desorden de la información personal. Ofrece una captura ultrarrápida de notas combinada con un sistema de etiquetado dinámico y flujos visuales que permiten al usuario vaciar su mente al instante y organizar o procesar esa información después, sin perder el foco de lo que está haciendo en el momento.

## 2. Usuario Objetivo y Caso de Uso Diario
*   **Perfil:** Estudiantes, desarrolladores, creadores de contenido y profesionales que trabajan en entornos dinámicos y necesitan gestionar su carga cognitiva.
*   **Día a día con la App:**
    *   **Mañana:** El usuario abre NoteFlow de camino al trabajo/clase y apunta rápidamente tres ideas que le han venido a la mente usando la interfaz limpia de la app.
    *   **Tarde:** Durante su jornada, usa los flujos o categorías (ej. *Personal*, *Proyectos*, *Estudios*) para clasificar esas notas rápidas, convirtiendo algunas en tareas pendientes y archivando otras.
    *   **Noche:** Revisa el "Flow" (el feed principal) para limpiar las notas temporales del día y dejar la mente despejada para el día siguiente.

## 3. Funcionalidades Principales (MVP - Versión 1)
*   **Captura Rápida:** Pantalla principal optimizada para crear notas de texto en un solo toque, sin campos obligatorios iniciales.
*   **Organización por Flujos/Etiquetas:** Creación y asignación de etiquetas personalizadas para filtrar las notas de forma instantánea.
*   **Estados de Nota:** Poder marcar una nota como "Activa", "Archivada" o convertirla en "Tarea Pendiente" (Checklist).
*   **Persistencia Local:** Almacenamiento seguro en el dispositivo para que la app funcione 100% offline desde el primer momento.
*   **Buscador en Tiempo Real:** Filtrado de notas por texto o etiquetas con respuesta inmediata.

## 4. Funcionalidades Opcionales (Futuras Versiones)
*   **Sincronización Cloud:** Respaldar y sincronizar notas entre múltiples dispositivos (iOS / Android / Web).
*   **Notas de Voz con Transcripción:** Capturar audio y convertirlo a texto automáticamente mediante IA local o API externa.
*   **Recordatorios Inteligentes:** Notificaciones push basadas en tiempo o geolocalización para notas convertidas en tareas.
*   **Modo Markdown Extendido:** Soporte para formato de texto enriquecido, bloques de código y enlaces entre notas.
